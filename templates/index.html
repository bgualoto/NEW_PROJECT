Información

    Se pretende realizar el análisis de sentimientos de textos en el idioma kichwa mediante un sistema web. Para dicho sistema, primero fue necesario tomar los textos que tenemos como entrada y pasarlos por el procesamiento natural del lenguaje, de manera que todo el texto se encuentre bajo un mismo estándar y sea más fácil de analizarlo, además, se hará uso del método de la distancia de Jaccard y de Coseno Vectorial para poder identificar que sentimiento esta produciendo el texto introducido ya sea positivo, negativo o neutro, para llevar a cabo este problema se propone un modelo de clasificación.
